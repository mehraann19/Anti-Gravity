# Security Policy

## Supported Versions

We actively maintain and provide updates for the following versions of the **Antigravity Hardened Ruleset Suite**:

| Version | Supported |
| :--- | :--- |
| **v2.0.x** (Active Release) | Yes |
| **v1.x.x** | No |

## Reporting a Vulnerability

If you discover a security issue or vulnerability related to the switcher script or installation routines:

1. **Private Disclosure**: Please do not open a public GitHub issue. Send a report directly to the repository maintainer.
2. **Details**: Provide a detailed description of the issue, environment setup, and steps to reproduce.
3. **Response Timeline**: We will acknowledge receipt of your report within 48 hours and coordinate a mitigation patch.
